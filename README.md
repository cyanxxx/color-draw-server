# color-draw-server
这是color-draw的server部分代码

### 对应前端页面
https://github.com/cyanxxx/colorDraw

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9090
npm run dev

# build for production with minification
npm run build

# start for production with minification
npm run start
```
